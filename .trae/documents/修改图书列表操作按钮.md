# 修改图书列表操作按钮

## 目标
将图书列表中的图标展示替换为三个可点击的按钮：编辑、删除、看书，并绑定点击事件。

## 修改内容

### 1. 修改 `index.vue` 模板部分
- 替换原来的 actions 图标展示
- 添加三个按钮：编辑、删除、看书
- 为每个按钮绑定点击事件，并传入 bookList 的 id

### 2. 修改 `index.vue` 脚本部分
- 添加三个点击事件处理函数：`handleEdit`、`handleDelete`、`handleRead`
- 每个事件处理函数接收 bookId 作为参数，并打印出来

## 具体修改步骤

1. **修改模板中的操作区域**：
   - 替换原来的 `v-for="action in actions"` 循环
   - 添加三个 `Button` 组件
   - 为每个按钮设置不同的类型和图标
   - 绑定对应的点击事件

2. **添加事件处理函数**：
   - 在 `<script>` 标签中添加三个函数
   - 每个函数接收 `bookId` 参数
   - 函数体中打印 `bookId`

3. **调整样式**：
   - 确保按钮布局合理
   - 调整按钮间距和大小

## 预期效果
- 图书列表中的每个条目右侧会显示三个可点击的按钮
- 点击任意按钮会在控制台打印出对应的图书 id
- 按钮样式符合整体设计风格

## 代码示例

### 模板修改
```vue
<!-- 替换原来的 actions 循环 -->
<div :class="`${prefixCls}__action`">
  <a-button type="primary" size="small" @click="handleEdit(item.id)">
    编辑
  </a-button>
  <a-button type="danger" size="small" @click="handleDelete(item.id)">
    删除
  </a-button>
  <a-button size="small" @click="handleRead(item.id)">
    看书
  </a-button>
</div>
```

### 脚本修改
```typescript
// 添加事件处理函数
const handleEdit = (bookId: string) => {
  console.log('编辑图书:', bookId);
};

const handleDelete = (bookId: string) => {
  console.log('删除图书:', bookId);
};

const handleRead = (bookId: string) => {
  console.log('查看图书:', bookId);
};
```
